# SrisVoice 🎙️
### AI-Powered English Speaking Practice App

---

## ⚡ 30 Minute Mein Live Karo

### Step 1 — Computer Pe Chalao (Local)
```bash
# 1. Node.js install karo: nodejs.org
# 2. Yeh folder open karo terminal mein
npm install
npm run dev
# Browser mein khulega: http://localhost:5173
```

---

### Step 2 — GitHub Pe Upload Karo
```bash
# GitHub Desktop app use karo (easy):
# desktop.github.com se download karo

# Ya terminal se:
git init
git add .
git commit -m "SrisVoice first commit"
git branch -M main
git remote add origin https://github.com/AAPKA_USERNAME/srisvoice.git
git push -u origin main
```

---

### Step 3 — Vercel Pe Deploy Karo (FREE)
```
1. vercel.com pe jaao
2. "Sign Up" → "Continue with GitHub"
3. "New Project" → "srisvoice" repo select karo
4. "Deploy" click karo
5. 2 minute mein URL milegi:
   → https://srisvoice.vercel.app ✅
```

### Step 4 — WhatsApp Pe Share Karo
```

 banaya hai — AI se practice kar sakte ho!
 Try karo: https://srisvoice.vercel.app"
```
